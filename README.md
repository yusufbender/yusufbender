## What I'm Building

### 🧠 BenderEdge
**Multi-agent AI stock research platform**

5 specialized agents analyze stocks in parallel and vote on a BUY/SELL/HOLD verdict using weighted confidence scoring.

- Researcher Agent → live news analysis via NewsAPI + LLM
- Quant Agent → RSI, MACD, Bollinger Bands, SMA, volume
- Sentiment Agent → headline tone scoring (-10 to +10)
- BenderQuant ML Agent → on-the-fly XGBoost training, backtest, equity curve
- Portfolio Agent → weighted voting, confidence scoring

**Stack:** FastAPI · LangChain · Ollama (Qwen2.5:7b) · XGBoost · Next.js · SSE

👉 [github.com/yusufbender/BenderEdge](https://github.com/yusufbender/BenderEdge)

---

### 📈 BenderQuant
**XGBoost financial classification model**

Buy/no-buy signal system trained on multi-ticker OHLCV data with feature engineering, SMOTE oversampling, and GridSearch hyperparameter tuning.

Features: RSI · MACD · Bollinger Bands · EMA · SMA · Volume normalization · Trend crossover

Includes: cross-validation · model comparison (XGBoost vs RF vs LightGBM vs CatBoost) · backtest engine · Sharpe ratio · Max Drawdown · CAGR

👉 [github.com/yusufbender/benderquant](https://github.com/yusufbender/benderquant)
